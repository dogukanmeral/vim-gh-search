*vim-gh-search.txt* vim-gh-search is a plugin that searches code parts in Github using your default web-browser.

USAGE                                                    *vim-gh-search-usage*

Searching selected text in visual mode:
	:GhSearchSelected
	Gets selected text in visual mode from register, searches the text
	with specified parameters in .vimrc.
	(shortcut: gs in visual mode by default)

Searching manually with parameters:
	:GhSearch
	Switches to command prompt where you specify text and arguments in a
	more like "bash" style and redirects to default browser.
	(Reason for command prompt is to make script portable to other text
	editors later on)

==============================================================================
ARGUMENTS AND OPTIONS                            *vim-gh-search-args-and-opts*

Positional Arguments:
  text             Code part to search

Options:
  -h, --help       show this help message and exit
  --user USER      Filter by user
  --org ORG        Filter by organization
  --repo REPO      Filter by repository
  --lang LANG      Filter by language
  --path PATH      Filter by path
  --symbol SYMBOL  Filter by symbol
  --archived       Search in archived repositories
  --vendored       Search in vendored repositories
  --generated      Search in generated repositories
